Developer-Friendly API

Integrate MediaAPI into your own application with our simple REST API

POST /api/media/upload 
curl -X POST https://api.mediaapi.com/api/media/upload \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -F "file=@image.jpg" \
  -F "category=image" \
  -F "title=My Photo"
POST /api/licenses/request 
curl -X POST https://api.mediaapi.com/api/licenses/request \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "mediaId": "media_123",
    "licenseType": "commercial",
    "price": 100
  }'
POST /api/licenses/:id/pay 
curl -X POST https://api.mediaapi.com/api/licenses/123/pay \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json"
